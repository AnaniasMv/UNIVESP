Essa aula é destinada a aprender tabela hash baseado em um problema pratico:
Em uma faculdade é criado um banco de dados com o RA e o Nome de todos os alunos.
Como é possivel criar uma busca eficiente neste Banco de Dados?
É isto que veremos nos códigos contidos nesta pasta

hash.h e hash.cpp é onde estará nossa classe hash
main hash é onde estará a função principal
aluno h e aluno.cpp é onde estará nossa classe aluno
